# Generate a table of content for markdown file
## Use
Just run
```
python md_toc.py filename.md
```

if you add it to your bashrc:
```
echo "alias toc='python <path-to-script>/md_toc.py'" >> ~/.bashrc
source ~/.bashrc
toc filename.md
```
